# icmp-sweep-tool
Simple bash script to do an ICMP sweep on a local /24 network

## Usage
Save the script and run the script with the network ID supplied as an argument:
```bash
./ipsweep.sh <network ID>
```
example:
```bash
./ipsweep.sh 192.168.1
```

## Dependencies
Runnable on Linux/WSL environments. Must have ping capability.
